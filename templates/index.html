Introduction

Simplify your PowerPoint workflow. Type in your search words and get images and translations downloaded automatically.

How to use

  • Go to the Downloader tab.
  • Select the translation language.
  • Enter the words that you would like to find images and translations for.
  • Select the number of images that you would like to download for each search.
  • You can rearrange the input boxes by placing your curser just below an input box.
  • The curser icon will change. You can click and drag to rearrange the input box.
  • After you input a word, you can hit Enter to create more input boxes
  • Click submit when ready.
  • Firefox browser will open and automatically get the translations and images.
  • A folder called 'multiImageSearchDownloads' will be created in your Local Disk Drive(C:).
  • The images and a text file containing the translations will be downloaded into this folder.
  • When complete, the website will display Download complete!

Notes on use

  • Selenium, which is a suite of tools for automating web browsers, is used for automated web browsing.
  • The browsers used are: Firefox as well as Firefox Developer Edition.
  • Bing image search is used to find images and Papago.naver.com is used for translations.
  • The downloaded images are large and have a wide layout (good for PowerPoint presentations). The images are free to share and use.
